# Hermes Branding Permission

**Recorded:** 2026-07-16
**Project:** Hermex Android
**Upstream:** `hermes-webui`

The Hermex project owner confirms that Hermex has permission from the
`hermes-webui` project to use the Hermes branding currently shipped by the Android
app, including the Hermes wordmark and the upstream-derived launcher-icon artwork.

The owner further confirms that Hermex is developed hand in hand with
`hermes-webui`. Store and product copy may accurately describe that collaborative
relationship. This confirmation does not, by itself, designate Hermex as the
official Android app or authorize claims beyond the collaboration and branding use
described above.

This file records the project owner's attestation made during the Play-readiness
review. If the original upstream approval is available as an issue, email, or other
durable record, its reference should be added here for provenance.
