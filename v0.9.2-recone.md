- Top bar: `TopAppBar` with title + back/search/refresh actions

- Loading: centered `CircularProgressIndicator`
- Toast: `Toast.makeText` (workspace actions)
- Clipboard: `ClipboardManager` + `ClipData` (copy path)

## 7. Proposed Action Behavior

### "Open Raw" (WorkspaceScreen)
- **Where:** Entry row overflow menu, third option after "Copy path" and "Rename"
- **Icon:** `Icons.Filled.OpenInNew` or similar
- **Behavior:** Calls `api.fileRaw(sessionId, path)` to get raw content bytes, then opens in an in-app viewer or passes to an external viewer via `Intent.ACTION_VIEW`
- **Success:** Opens content in external app or inline raw viewer
- **Error:** Toast message if file cannot be read
- **Permissions:** None ‚ÄĒ uses ContentProvider URI via `FileProvider` or returns raw content inline
- **Tests:** ViewModel unit test verifies API method is called; ADB smoke test for UI

### "Open in..." / "Share" (WorkspaceScreen)
- **Where:** Entry row overflow menu
- **Behavior:** Downloads raw bytes via `fileRaw` API, writes to a temp file, creates `FileProvider` URI, fires `Intent.ACTION_VIEW` or `Intent.ACTION_SEND`
- **Success:** System chooser opens with available apps
- **Error:** Toast if file cannot be downloaded
- **Permissions:** `FileProvider` already configured or add to manifest
- **Tests:** Primarily manual/ADB verification

### Copy name/id (detail screens)
- **Where:** Detail screen top-bar overflow menu or row's icon button
- **Behavior:** `ClipboardManager.setPrimaryClip(ClipData.newPlainText(...))`
- **Success:** Toast "Copied to clipboard"
- **Error:** None
- **Tests:** Unit testable ‚ÄĒ ViewModel emits copy event, StateFlow asserts

### Search/filter (TasksScreen, ProjectsScreen)
- **Where:** Top bar search icon toggle (reuse SessionListScreen pattern)
- **Behavior:** On toggle, show `TextField` in top bar, filter list by name/title
- **Success:** List filters as user types
- **Error:** No-error fallback: "No matching tasks/projects."
- **Tests:** ViewModel test with search query ‚Üí filtered items assertion
