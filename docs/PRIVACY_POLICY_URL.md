# Privacy Policy URL

The canonical Hermex Android privacy-policy URL is:

https://computerbyte.github.io/hermex-android/privacy/

Source: `docs/privacy/index.md`

GitHub Pages must publish the `/docs` directory from the repository's default
branch. After the page is enabled or updated, verify that the canonical URL is
publicly reachable over HTTPS and returns the current policy before submitting it
to Google Play.
